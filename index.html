🎯 CLOSING-LINE VALUE — the real edge test

For every bet the model would place, we compare the price it would have taken (the opening odds snapshot) with the closing line (last snapshot before kickoff). Beating the close — positive CLV — is the one signal that genuinely separates skill from luck, so this is the scoreboard that matters.

So far: 3 snapshot(s) since 2026-06-20, 53 fixtures tracked, 9 with an open+close pair. The live updater snapshots the closing line just before each kick-off, so this lights up on its own as games are priced and played — beating the closing line is the only honest proof of edge.
